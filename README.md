# Facial Recognition Attendance System

## Overview
This project is a **Facial Recognition Attendance System** built using **OpenCV**, **NumPy**, and **Tkinter**. The system recognizes faces via a webcam, registers the faces, and records attendance. It provides an admin dashboard where faces can be registered, recognized, and attendance can be viewed.

## Features
- **Face Registration**: Allows users to register their faces with a name.
- **Face Recognition**: Real-time face recognition via webcam and automatic attendance marking.
- **Attendance Records**: Tracks attendance and shows the list of registered names and their attendance time.
- **Admin Dashboard**: Simple Tkinter-based GUI to interact with the system.

## Requirements
Before running the project, ensure you have the required libraries installed.

- **OpenCV** for computer vision tasks.
- **NumPy** for array manipulation.
- **Tkinter** for GUI interaction.
